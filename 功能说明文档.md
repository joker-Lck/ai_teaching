# 多模态 AI 教学智能体 - 功能文档

## 📋 项目简介

基于 Kimi（月之暗面）大模型的校园 AI 教学助手，为教师提供智能化、多模态的教学支持工具。

**核心优势：**
- 🤖 采用 Kimi 大模型，支持超长上下文理解
- 🎯 专为 K12 教育场景设计
- 💡 全栈 AI 能力集成
- 🚀 三库分离架构，数据安全有保障
- 💾 全量数据持久化，刷新不丢失
- 📄 统一 JSON 格式存储，便于管理
- 🔍 **RAG 优先策略**：提问时自动从知识库检索，减少 AI 幻觉

---

## 🎯 核心功能

### 1️⃣ 意图沟通（智能答疑）

**三大场景标签页：**

#### 📖 课前预习
- **功能**：学生预习阶段的智能答疑
- **输入方式**：文字输入问题
- **AI 能力**：精准匹配教学问题，提供详细解答
- **🔍 RAG 优先策略**（新增）：
  - 第一步：查询历史问答记录（相似度 > 80% 直接返回）
  - 第二步：检索 RAG 知识库（93 本电子书）
  - 第三步：基于知识库内容生成答案（减少幻觉）
  - 显示参考来源（如“参考了 3 本书籍”）
- **适用场景**：学生自主预习、课前准备
- **📝 聊天记录**：自动保存所有问答记录

#### 🎯 课中互动
- **功能**：课堂实时互动工具
- **子功能**：
  - **📋 班级管理**
    - 上传班级名单（支持 Excel/CSV/TXT）
    - 手动输入学生名单
    - 显示班级总人数
  - **🎲 随机点名**
    - 一键随机抽取学生
    - 大屏幕高亮显示被选中姓名
    - 自动记录互动数据
    - **📊 互动记录管理**
      - 查看所有点名记录（时间、班级、学生）
      - 批量删除选中的互动记录
      - 一键清空所有互动记录
  - **💬 实时提问**
    - 文字提问模式
    - 语音提问模式（按住说话）
    - **💾 聊天记录保存**（新增）
      - 自动保存所有问答内容
      - 支持搜索历史记录
      - 支持导出为 TXT/JSON 格式
- **适用场景**：课堂教学、小组协作、课堂讨论

#### 📝 课后辅导
- **功能**：作业批改与个性化辅导
- **子功能**：
  - **📸 作业拍照批改**
    - 上传作业图片（jpg/png/jpeg）
    - AI 智能批改并给出反馈
    - 显示正确率、薄弱点、学习建议
  - **💬 作业问题答疑**
    - 文字输入作业中的疑问
    - AI 详细讲解错题
- **适用场景**：家庭作业、课后练习、个性化辅导

**📎 教学资料上传与解析**
- **支持格式**：PDF、Word (doc/docx)、PPT (ppt/pptx)、TXT
- **AI 解析内容**：
  - 📚 知识点提炼（核心概念、重点难点）
  - 🎯 教学目标（知识目标、能力目标、素养目标）
  - 📝 典型例题（3-5 道代表性题目及解析）
  - 💡 教学建议（推荐教学方法、活动设计）
  - ⏰ 课时安排（建议学习时长、进度规划）
  - 🔗 拓展资源（相关知识点链接、延伸阅读）
- **🗑️ 文件管理**（新增）
  - 批量选择并删除已上传的文件
  - 一键清空所有文件
  - 按学科分类展示（语文、数学、英语等）
- **💾 记录保存**（新增）
  - 所有 AI 解析内容自动保存
  - 可在聊天记录中查看和搜索

### 2️⃣ AI 课件生成

**功能描述**：根据主题和学科自动生成完整 PPT 课件

**使用流程**：
1. 输入课件主题（如“函数的单调性”）
2. 填写年级（支持任意年级，如：高一、八年级、大学一年级等）
3. （可选）上传参考资料
4. 点击“开始生成课件”

**🔍 RAG 优先策略**（新增）：
- **第一步**：根据主题自动识别学科（AI 智能识别）
- **第二步**：从 RAG 知识库检索相关教学资料（93 本电子书）
- **第三步**：基于知识库内容生成课件大纲和 PPT 结构
- **第四步**：显示参考来源（如“参考了 3 篇教学资料”）

**AI 生成内容**：
- 📊 教学需求分析
- 🎯 教学目标
- 📝 重点难点
- 📚 知识点列表
- 💡 典型例题
- 🎨 教学活动设计
- 📄 **完整 PPT 课件（10-15 页，JSON 格式）**

**输出形式**：
- 📊 PPT 大纲（左侧显示）
- 📝 教案详情（右侧文本框，可编辑）
- 📄 **PPT 课件预览**（折叠展示每一页）
- 📥 **PPT 下载按钮**
  - ✅ **真实 PPT 文件** (.pptx) - Kimi 生成 JSON → Python 转换为 PowerPoint
  - ✅ 可在 PowerPoint/WPS 中直接打开编辑
- 📥 **Word 教案下载按钮**（.docx 格式，配套详细教案）

### 3️⃣ AI 学情分析

**双模式分析：**

#### 👤 单个学生模式
- 输入学生姓名
- 生成个人学习报告
- 个性化学习建议

#### 👥 全班评估模式
- 输入班级名称（如"高一 (3) 班"）
- 设置班级总人数
- 生成班级整体学情报告

**📁 数据上传**
- **支持格式**：Excel (xlsx/xls)、CSV、PDF、Word、TXT、图片
- **文件类型**：成绩单、作业统计表、考试分析、学生作品等
- **AI 处理**：自动读取并分析上传的文件内容

**📊 统计数据展示**
- 问题总数（基于互动记录）
- 平均正确率（初始 0%，AI 分析后更新）
- 学习时长（初始 0 分钟，累计显示）
- 最近互动记录（显示最近 5 条）
- 薄弱知识点列表
- **🗑️ 聊天记录管理**（新增）
  - 批量删除选中的问答记录
  - 一键清空所有聊天记录
  - 显示每条记录的预览（场景标签 + 问题摘要）

**🤖 AI 生成学情报告（7 大维度）**
1. 📊 **整体情况概览** - 平均分、优秀率、及格率等关键指标
2. 📈 **成绩分布分析** - 分数段统计、正态分布分析
3. 🎯 **知识点掌握情况** - 优势知识点、薄弱知识点 TOP5
4. 👥 **学生分层分析** - 学优生、中等生、学困生比例及特点
5. 📉 **典型问题分析** - 错误率高的题目类型和原因
6. 💡 **个性化教学建议** - 针对不同层次学生的具体建议
7. 📋 **后续教学计划** - 重点讲解内容、练习安排

**📈 数据可视化建议**
AI 会提示如何制作：
- 成绩分布直方图
- 知识点掌握雷达图
- 学生分层饼图
- 成绩变化趋势图

### 4️⃣ 知识库管理（RAG 知识库）

**功能**：展示和管理教学资源库，支持全文检索和智能匹配

**📊 知识库统计**（当前状态）：
- **RAG 文档总数**：93 本电子书
- **覆盖学科数**：5 大类（AI, C++, Extra, Math, Python）
- **知识点数量**：AI 自动提取
- **平均使用次数**：持续更新中

**📚 预置书籍内容**：

| 类别 | 内容描述 | 文档数 |
|------|----------|--------|
| AI | 人工智能、深度学习、机器学习等 | ~20 |
| C++ | C++ 编程基础、算法与数据结构 | ~18 |
| Extra | 其他计算机相关书籍 | ~15 |
| Math | 数学分析、线性代数、概率论等 | ~20 |
| Python | Python 编程、数据分析、Web 开发等 | ~20 |

**📚 文档收纳与展示**
- 按学科分类展示（11 个学科）
- 显示每个学科的文档数量
- 支持文档详情查看和下载
- **🔍 全文检索**：支持关键词搜索，快速定位相关文档
- **📂 学科筛选**：可按学科分类筛选文档
- **🗑️ 文档管理**：支持删除单个或批量删除文档

**🤖 AI 智能解析**
- 自动解析上传的文档内容（PDF/Word/PPT/TXT）
- 提取知识点标签
- 生成 AI 摘要
- 支持全文检索（FULLTEXT 索引）

**🔍 RAG 优先策略**（核心创新）：
- **智能答疑时**：先查 QA 数据库 → 再查 RAG 知识库 → 最后调用 AI
- **课件生成时**：先从知识库检索相关教学资料 → 基于资料生成课件
- **优势**：
  - ✅ 减少 AI 幻觉，提高答案准确性
  - ✅ 节省 Token 消耗，降低成本
  - ✅ 显示参考来源，增强可信度

**📊 统计信息**
- RAG 文档总数
- 知识点数量
- 平均使用次数
- 覆盖学科数

**💾 持久化存储**
- **三层存储架构**：
  1. **RAG 数据库**：`ai_rag_knowledge` 知识库数据库（全文检索、学科分类、知识点管理）
  2. **JSON 文件**：本地 JSON 格式备份（便于查看和迁移）
  3. **Session State**：前端展示缓存
- **自动去重**：上传时自动检测重复文档，避免重复存储
- **刷新不丢失**：数据库恢复机制，刷新页面自动加载历史文档

**📤 支持格式**
- PDF、Word (doc/docx)、PPT (ppt/pptx)、TXT、Markdown

## 🗄️ 数据库架构

### 三库分离架构

**1️⃣ 主业务数据库：`ai_teaching_assistant`**
- **用途**：核心业务数据
- **包含表**：
  - `users` - 用户信息表
  - `classes` - 班级信息表
  - `students` - 学生信息表
  - `courseware` - 课件资源表（JSON 格式存储完整课件数据）
  - `learning_analysis` - 学情分析记录表

**2️⃣ 答疑专用数据库：`ai_qa_records`**
- **用途**：智能答疑记录
- **包含表**：
  - `qa_records` - 问答记录表（包含问题、答案、场景、时间等）
  - `qa_sessions` - 会话历史表
  - `interaction_records` - 互动记录表（随机点名记录）

**3️⃣ RAG 知识库数据库：`ai_rag_knowledge`**
- **用途**：知识库文档管理
- **包含表**：
  - `knowledge_documents` - 知识文档表（JSON 格式存储完整文档数据）
  - `knowledge_points` - 知识点关联表
  - `document_categories` - 文档分类表
  - `document_category_relation` - 文档分类关联表
  - `document_usage_log` - 文档使用日志表

**✨ 架构优势**：
- 📦 数据隔离，互不影响
- 🔍 RAG 库支持全文检索（FULLTEXT 索引）
- 📊 统一 JSON 格式存储，便于扩展
- 🔄 自动备份机制（数据库 + 本地 JSON 文件）
- 🛡️ 支持独立备份和恢复

## 🔧 常见问题

### Q: 如何更换 API Key？
A: 在代码开头修改 `DEFAULT_API_KEY` 变量即可。

### Q: 支持哪些文件格式上传？
A: 
- 课件解析：PDF、Word、PPT、TXT
- 学情分析：Excel、CSV、PDF、Word、TXT、图片
- 班级名单：Excel、CSV、TXT
- 知识库文档：PDF、Word、PPT、TXT

### Q: 如何查看学生的互动记录？
A: 在"学情分析"模块可以看到最近的互动记录和问题统计。

### Q: 随机点名如何保证公平性？
A: 使用 Python 的 `random.choice()` 函数实现真正的随机抽取。

### Q: AI 生成的内容可以保存吗？
A: 可以！当 AI 回复超过 100 字时，会自动显示"📥 下载 AI 回复为文本文件"按钮，一键保存为 .txt 文件。

### Q: AI 回复中的链接可以直接访问吗？
A: 是的，AI 回复中的链接会被自动识别并显示在"📎 相关链接"区域，点击即可访问。

### Q: 如何切换背景颜色？
A: 当前版本采用简洁的纯白色背景，如需个性化主题可自行修改 CSS 样式。

### Q: 互动记录可以删除吗？
A: 可以！在"课中互动 → 随机点名"模块底部，有"互动记录管理"区域，支持批量删除和清空操作。

### Q: 聊天记录太多可以清理吗？
A: 可以！在"智能答疑"模块顶部有"聊天记录管理"区域，支持多选删除和一键清空。

### Q: 上传的文件可以删除吗？
A: 可以！在"知识库管理"模块，可以选择要删除的文档或清空所有文档。文档会从数据库和本地 JSON 文件中同步删除。

### Q: 知识库支持全文检索吗？
A: 支持！RAG 知识库使用 MySQL FULLTEXT 索引，支持中文全文检索，可以快速搜索文档内容。

### Q: 如何查找之前的聊天记录？
A: 在"聊天记录管理"区域的搜索框中输入关键词，系统会自动筛选包含该关键词的问题和答案。

### Q: 聊天记录可以导出保存吗？
A: 可以！点击"导出记录"按钮，选择导出格式（TXT 或 JSON），然后点击"📤 导出聊天记录"按钮即可下载。

### Q: 导出的聊天记录包含哪些内容？
A: 
- **TXT 格式**：包含问题、AI 回答、时间、场景等信息，格式清晰易读
- **JSON 格式**：结构化数据，便于程序处理和数据迁移
