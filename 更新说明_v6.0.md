## 🚀 重大升级

### 架构重构
从 **Streamlit 单体应用** 升级为 **Next.js + FastAPI 前后端分离架构**

#### 前端技术栈
- ✅ Next.js 14（React 服务端渲染框架）
- ✅ React 18 + TypeScript
- ✅ Tailwind CSS 3（原子化样式）
- ✅ Framer Motion（交互动画）
- ✅ Zustand（状态管理）
- ✅ Recharts（数据可视化）
- ✅ Socket.IO Client（实时通信）

#### 后端技术栈
- ✅ FastAPI 0.110+（高性能 API 框架）
- ✅ Uvicorn（ASGI 服务器）
- ✅ JWT Token 认证
- ✅ SSE 流式输出
- ✅ WebSocket 支持
- ✅ 自动 API 文档（Swagger UI / ReDoc）

## 📝 使用变化

### 启动方式

**旧版本（v5.0）**：
```bash
streamlit run streamlit_001.py
```

**新版本（v6.0）**：
```bash
# Windows（推荐）
启动v6.bat

# 或手动启动
# 终端1 - 后端
python -m uvicorn backend.main:app --host 0.0.0.0 --port 8000 --reload

# 终端2 - 前端
cd frontend
npm run dev
```

### 访问地址

| 服务 | 旧版本 | 新版本 |
|------|--------|--------|
| 前端应用 | http://localhost:8501 | http://localhost:3000 |
| 后端 API | - | http://localhost:8000 |
| API 文档 | - | http://localhost:8000/api/docs |

### 环境要求

**新增要求**：
- Node.js 18+
- npm 9+ 或 yarn 1.22+

**保留要求**：
- Python 3.8-3.11
- MySQL 8.0+

## 🔄 迁移指南

### 从 v5.0 升级到 v6.0

1. **备份数据**
   ```bash
   # 备份数据库
   mysqldump -u root -p ai_teaching_assistant > backup_v5.sql
   mysqldump -u root -p ai_qa_database > backup_qa_v5.sql
   mysqldump -u root -p ai_rag_knowledge > backup_rag_v5.sql
   ```

2. **安装新依赖**
   ```bash
   # 后端依赖
   pip install -r backend/requirements.txt
   
   # 前端依赖
   cd frontend
   npm install
   ```

3. **初始化数据库**
   ```bash
   python init_db.py
   python init_qa_db.py
   python init_rag_db.py
   ```

4. **启动新版本**
   ```bash
   启动v6.bat
   ```
