# 用户认证系统使用说明 (v6.0)

## 📋 功能概述

多模态 AI 教学智能体 v6.0 采用前后端分离架构，用户认证通过 FastAPI 后端提供的 REST API 实现。

### 核心功能

- ✅ **用户注册**：支持教师和学生两种角色
- ✅ **用户登录**：用户名 + 密码登录，返回 JWT Token
- ✅ **游客模式**：无需注册即可体验基本功能
- ✅ **密码加密**：使用 SHA-256 + Salt 加密存储
- ✅ **角色管理**：教师、学生、管理员三种角色
- ✅ **Token 认证**：基于 JWT 的无状态认证
- ✅ **退出登录**：清除本地 Token

## 🔐 认证机制

### JWT Token 认证

v6.0 版本采用 JWT (JSON Web Token) 进行身份验证：

1. **登录流程**：
   ```
   用户输入账号密码 → POST /api/auth/login → 返回 JWT Token
   ```

2. **请求认证**：
   ```
   前端在请求头携带 Token → 后端验证 Token → 返回数据
   Authorization: Bearer <token>
   ```

3. **Token 有效期**：
   - Access Token: 24 小时
   - Refresh Token: 7 天

### 密码加密

- **算法**：SHA-256
- **Salt**：16 字节随机盐值
- **存储格式**：`salt$hash`
- **特点**：
  - 相同密码每次加密结果不同（因 salt 不同）
  - 防止彩虹表攻击
  - 防止密码泄露后被破解

### 密码要求

- 长度：至少 6 位
- 建议使用字母 + 数字组合
- 避免使用常见密码

## 📝 API 接口

### 认证相关接口

所有接口基础 URL：`http://localhost:8000/api/auth`

#### 1. 用户注册

**接口**：`POST /api/auth/register`

**请求体**：
```json
{
  "username": "test_teacher",
  "password": "123456",
  "email": "test@example.com",
  "role": "teacher"
}
```

**响应**：
```json
{
  "success": true,
  "message": "注册成功",
  "data": {
    "user_id": 1,
    "username": "test_teacher"
  }
}
```

#### 2. 用户登录

**接口**：`POST /api/auth/login`

**请求体**：
```json
{
  "username": "admin",
  "password": "admin123"
}
```

**响应**：
```json
{
  "success": true,
  "message": "登录成功",
  "data": {
    "access_token": "eyJhbGciOiJIUzI1NiIs...",
    "refresh_token": "eyJhbGciOiJIUzI1NiIs...",
    "token_type": "bearer",
    "user": {
      "id": 1,
      "username": "admin",
      "role": "admin"
    }
  }
}
```

#### 3. 获取当前用户信息

**接口**：`GET /api/auth/me`

**请求头**：
```
Authorization: Bearer <access_token>
```

**响应**：
```json
{
  "success": true,
  "data": {
    "id": 1,
    "username": "admin",
    "email": "admin@example.com",
    "role": "admin"
  }
}
```

#### 4. 修改密码

**接口**：`PUT /api/auth/password`

**请求头**：
```
Authorization: Bearer <access_token>
```

**请求体**：
```json
{
  "old_password": "old123",
  "new_password": "new456"
}
```

更多接口详情请查看 API 文档：http://localhost:8000/api/docs

## 💻 前端集成

### Zustand Store 示例

```typescript
// stores/authStore.ts
import { create } from 'zustand'

interface AuthState {
  user: User | null
  token: string | null
  isAuthenticated: boolean
  login: (username: string, password: string) => Promise<void>
  logout: () => void
}

export const useAuthStore = create<AuthState>((set) => ({
  user: null,
  token: null,
  isAuthenticated: false,
  
  login: async (username, password) => {
    const response = await fetch('/api/auth/login', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ username, password })
    })
    const data = await response.json()
    
    if (data.success) {
      set({
        user: data.data.user,
        token: data.data.access_token,
        isAuthenticated: true
      })
      localStorage.setItem('token', data.data.access_token)
    }
  },
  
  logout: () => {
    set({ user: null, token: null, isAuthenticated: false })
    localStorage.removeItem('token')
  }
}))
```

### API 请求拦截器

```typescript
// lib/api.ts
import axios from 'axios'

const api = axios.create({
  baseURL: process.env.NEXT_PUBLIC_API_URL || 'http://localhost:8000'
})

// 请求拦截器：自动添加 Token
api.interceptors.request.use((config) => {
  const token = localStorage.getItem('token')
  if (token) {
    config.headers.Authorization = `Bearer ${token}`
  }
  return config
})

// 响应拦截器：处理 Token 过期
api.interceptors.response.use(
  (response) => response,
  (error) => {
    if (error.response?.status === 401) {
      // Token 过期，跳转到登录页
      window.location.href = '/login'
    }
    return Promise.reject(error)
  }
)

export default api
```

## 💡 常见问题

### Q1: 忘记密码怎么办？

目前版本暂不支持密码找回功能，建议：
- 记住密码或使用密码管理器
- 联系管理员重置密码

### Q2: 游客模式有什么限制？

游客模式**只能使用智能答疑功能**，其他功能均不可用：

**可用功能**：
- ✅ 智能答疑（提问和查看AI回复）

**不可用功能**：
- ❌ 课件生成
- ❌ 学情分析
- ❌ 知识库管理

**其他限制**：
- 无法保存聊天记录
- 无法导出学习数据
- 无法使用个性化功能

### Q3: Token 过期如何处理？

前端会自动检测 Token 过期（401 错误），并：
1. 清除本地存储的 Token
2. 重定向到登录页面
3. 提示用户重新登录

### Q4: 如何刷新 Token？

使用 Refresh Token 接口：

**接口**：`POST /api/auth/refresh`

**请求体**：
```json
{
  "refresh_token": "eyJhbGciOiJIUzI1NiIs..."
}
```

### Q5: 数据库中的密码是明文吗？

不是！密码使用 SHA-256 + Salt 加密存储，即使数据库泄露也无法直接获取明文密码。

## 🔧 开发指南

### AuthService 类（后端）

主要方法：

```python
from services.auth_service import auth_service

# 注册
result = auth_service.register_user(username, password, email, role)

# 登录
result = auth_service.login_user(username, password)

# 验证 Token
user = auth_service.verify_token(token)

# 修改密码
result = auth_service.update_password(user_id, old_password, new_password)

# 获取用户信息
user = auth_service.get_user_by_id(user_id)
```

### 前端状态管理

使用 Zustand 管理认证状态：

```typescript
import { useAuthStore } from '@/stores/authStore'

function LoginPage() {
  const { login, user, isAuthenticated } = useAuthStore()
  
  const handleLogin = async () => {
    await login(username, password)
  }
  
  return (
    <div>
      {isAuthenticated ? (
        <p>欢迎, {user?.username}</p>
      ) : (
        <button onClick={handleLogin}>登录</button>
      )}
    </div>
  )
}
```
